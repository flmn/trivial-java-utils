# trivial-java-utils

## How to use

### For Gradle users

```
repositories {
    jcenter()
    maven { url 'https://jitpack.io' }
}
```

Include `https://jitpack.io` in repositories.

```
dependencies {
    ...
    implementation 'com.github.flmn:trivial-java-utils:$version'
    ...
}
```

Include `com.github.flmn:trivial-java-utils:$version` in dependencies.
